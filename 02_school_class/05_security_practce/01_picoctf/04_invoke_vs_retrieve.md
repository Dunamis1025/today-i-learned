# 04. Invoke vs Retrieve (Concept Understanding)

---

## 1. Invoke

### Definition
Invoke means to call, trigger, or execute something.

---

### In Programming
- Invoke a function = call and run it

Example:
You invoke a function to execute specific code.

---

### In Real Life
- Invoke emotion = trigger a feeling

Example:
This song invokes memories from my childhood.

---

### Key Idea
- Something is NOT active yet
- You make it happen

---

## 2. Retrieve

### Definition
Retrieve means to get something back.

---

### In Programming / IT
- Retrieve data = get stored data

Example:
The system retrieves data from the database.

---

### In Real Life
- Retrieve a lost item

Example:
I retrieved my file from the backup.

---

### Key Idea
- Something already exists
- You bring it back

---

## 3. Key Difference

| Concept   | Invoke                        | Retrieve                     |
|----------|-------------------------------|------------------------------|
| Meaning  | Trigger / Execute              | Get back / Bring back        |
| State    | Not started yet                | Already exists               |
| Action   | Start something                | Recover something            |
| Example  | Invoke a function              | Retrieve data                |

---

## 4. Simple Analogy

Invoke = Press "Play" ▶️  
Start something

Retrieve = Press "Download" ⬇️  
Bring something back

---

## 5. Summary

Invoke → Make something happen  

Retrieve → Bring something back
