# 10. Network Cabling Fundamentals – Coaxial & UTP

---

## 1. Coaxial Cable

### Structure
Coaxial cable consists of multiple layers:

- Copper conductor → transmits signals
- Plastic insulation → surrounds conductor
- Braided copper / metallic shield → protects signal and reduces interference
- Outer jacket → protects from physical damage

👉 Key idea:  
**Shielding reduces EMI (Electromagnetic Interference)**

---

### Connectors
- BNC
- N type
- F type

👉 Exam tip:  
**BNC = Coaxial cable**

---

### Usage
- Wireless installations → connects antennas (RF signal)
- Cable internet → used inside buildings

---

## 2. UTP Cable (Unshielded Twisted Pair)

### Structure
- 4 pairs of twisted copper wires
- Covered with flexible plastic sheath

👉 Small, cheap, easy to install

---

### Key Difference from Coaxial
- ❌ No shielding
- ✅ Uses twisting to reduce interference

---

## 3. Interference Handling (Very Important)

### 1. Cancellation
- Wires are paired
- Magnetic fields are opposite

👉 Result:
- Signals cancel interference
- Reduces EMI & RFI

---

### 2. Different Twist Rates
- Each wire pair has different number of twists

👉 Purpose:
- Reduce crosstalk (signal interference between wires)

---

### Key Concept

👉 **UTP = No shielding → uses twisting + cancellation**

---

## 4. Important Terms

- EMI → Electromagnetic Interference
- RFI → Radio Frequency Interference
- Crosstalk → Signal interference between wires
- Degradation → Signal quality gets weaker over time

---

## 5. UTP Standards

### Organizations
- TIA/EIA → cabling standards (TIA/EIA-568)
- IEEE → performance & categories

---

### TIA/EIA Defines
- Cable types
- Cable length
- Connectors
- Termination
- Testing methods

---

## 6. UTP Categories (VERY IMPORTANT)

| Category | Speed |
|--------|------|
| Cat3 | Voice / low data |
| Cat5 | 100 Mbps |
| Cat5e | 1 Gbps |
| Cat6 | up to 10 Gbps |
| Cat7 | 10 Gbps |
| Cat8 | 40 Gbps |

---

### Key Notes

- Cat5e → minimum standard today
- Cat6 → recommended for new buildings
- Higher category = higher bandwidth

---

## 7. Quick Comparison

| Type | Shielding | Use |
|------|----------|-----|
| Coaxial | Yes | Antenna, cable internet |
| UTP | No | LAN (most common) |
| STP | Yes | High interference environments |

---

## 8. Exam Summary

- Most common cable → **UTP**
- Antenna connection → **Coaxial**
- EMI/RFI reduction → **STP**
- BNC connector → **Coaxial**

---

## Final One-Line Summary

👉 **UTP = twisted wires for interference control**  
👉 **Coaxial = shielded cable for stable signal transmission**
