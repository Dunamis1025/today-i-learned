# GUI with Tkinter (Python)

This directory contains all Tkinter-related study materials and projects.
It is organized into two main sections: **GUI basics** and a **GUI project**.

The goal of this folder is to:
- Learn Tkinter step by step
- Practice core GUI concepts through small examples
- Apply learned concepts in a complete, functional project
