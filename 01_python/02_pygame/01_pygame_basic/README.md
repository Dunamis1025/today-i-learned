# Pygame Basics

This folder contains step-by-step Pygame examples created by following
the "Nado Coding (나도코딩)" Pygame tutorial series.

## What I Learned
- Creating a game window
- Drawing images and sprites
- Handling keyboard input
- Frame rate (FPS) control
- Collision detection using Rect
- Basic game loop structure

## Notes
- These are **guided learning exercises**
- Code was typed manually to understand each concept
- Not original game projects

Tutorial source: https://www.youtube.com/@nadocoding
