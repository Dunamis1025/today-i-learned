# 10. Physical Layer Basics

---

## 1. What is the Physical Layer?

The physical layer is the lowest layer of the OSI model.

- It is responsible for transmitting raw bits (0s and 1s)
- It defines how data is physically sent over the network
- It belongs to the Network Access layer in the TCP/IP model

👉 Without the physical layer, communication cannot occur.

---

## 2. Purpose of the Physical Layer

The physical layer ensures that devices can physically connect and transmit data.

Key functions:
- Transmit bits over a medium
- Define hardware specifications
- Control signal encoding

👉 It focuses on HOW data is transmitted, not WHAT data is.

---

## 3. Types of Physical Connections

### 1. Wired Connection
- Uses physical cables
- Examples: Ethernet cables (UTP, copper)
- Common in offices and homes

✔ Stable and fast  
❌ Limited mobility

---

### 2. Wireless Connection
- Uses radio waves
- Examples: Wi-Fi, mobile networks

✔ Flexible and convenient  
❌ Can be less stable

---

## 4. Wired Network Example

In many offices:
- Devices connect using cables
- A switch is used to connect multiple devices
- Data is transmitted through physical cables

👉 This is called a wired network.

---

## 5. Wireless Network Example

In wireless environments:
- Devices connect using Wi-Fi
- Data is transmitted using radio waves
- Devices must connect to:
  - Wireless Access Point (AP)
  - Wireless Router

👉 Wireless networks are widely used due to convenience.

---

## 6. Key Comparison

| Type       | Medium        | Pros                | Cons               |
|------------|--------------|---------------------|--------------------|
| Wired      | Cable        | Stable, fast        | Limited mobility   |
| Wireless   | Radio waves  | Flexible, convenient| Less stable        |

---

## 7. Key Concepts Summary

- Physical layer = transmission of bits
- Communication requires a physical connection
- Two main types: wired and wireless
- Wired uses cables, wireless uses radio waves
- Devices need proper connection (switch or AP)

---

## 🔥 One-Line Summary

Physical Layer = How data is physically transmitted (cable or wireless)
