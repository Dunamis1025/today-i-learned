hardware, operating systems, networking fundamentals, mobile devices, security, and
IT professional/customer service skills.

studied, with a short descriptive filename summarizing its contents.

## How These Notes Are Organized

- Each `.md` file is a self-contained set of Q&A-style notes on a specific topic cluster.
- Notes generally follow a **question → answer → plain-language explanation** format,
  aimed at reinforcing concepts for CompTIA A+ / IT Essentials–style exam questions.
- Topics build cumulatively: hardware fundamentals → PC assembly & troubleshooting →
  networking → mobile devices & printers → virtualization/cloud → Windows OS →
  Linux/macOS/mobile OS → security → IT professional/customer-service skills.
