## 1. Optical Media Types (1.2.9)

| Suffix | Media | Write Behavior |
|--------|-------|----------------|
| `-R` | CD (Compact Disc) | Write once |
| `RW` | CD (Compact Disc) | Write many (rewritable) |
| `+/-RW` | DVD | Write many (rewritable) |
| `+/-R` | DVD | Write once |
| `-RE` | Blu-ray | Write many (rewritable) |
| `R` | Blu-ray | Write once |
| `ROM` | All optical media | Pre-recorded only (read-only) |

### Key Rules
- **R / -R / +R** → Write **once** (cannot be erased)
- **RW / -RW / +RW / -RE** → Write **many** times (erasable)
- **ROM** → Read-only; factory pre-recorded (e.g., movie DVDs, game discs)
- **-RE** is the Blu-ray specific term for "Re-writable"

## 2. Video Ports and Cables (1.2.10)

| Port | Color | Signal Type | Key Features |
|------|-------|-------------|--------------|
| **VGA** | Blue | Analog only | 3 rows, 15 pins; also called DE-15 / HD-15; legacy |
| **DVI** | White | Digital + Analog | Up to 24 pins; fading out in favor of HDMI |
| **HDMI** | Black | Digital | Developed for HD TVs; widely used in computers |
| **DisplayPort** | Black | Digital | For high-end PCs and home theater displays |
| **Thunderbolt 1/2** | White (Apple) | Digital | High-speed peripherals; transmits video via DisplayPort protocol |
| **Thunderbolt 3** | Black | Digital | Uses USB-C connector; 2× bandwidth of TB2; supports dual 4K monitors |
| **RCA** | Yellow/Red/White | Analog | Yellow = video; Red/White = left/right audio |

### DVI Subtypes
- **DVI-D** → Digital only
- **DVI-A** → Analog only
- **DVI-I** → Digital + Analog (most common alongside DVI-D)

## 3. Adapters and Converters (1.2.10)

| Component | Function | Example |
|-----------|----------|---------|
| **Adapter** | Physically connects one technology to another | DVI to HDMI adapter |
| **Converter** | Connects AND translates signals between technologies | USB 3.0 to SATA converter |

> **Key difference:** An adapter is purely physical; a converter also **translates the signal**.

## 4. IT Technician Toolkit (1.2.11)

### ESD (Electrostatic Discharge) Safety
- **Anti-static wrist strap** – worn on wrist/ankle; alligator clip connects to ground (case or outlet)
- **Anti-static mat** – grounding for the workspace

### Cleaning Tools
- **Compressed air can** – removes dust from CPU heatsink, fans, GPU; prevents heat-related errors

### Screwdrivers
- **Phillips head** – most common; should always be in toolkit
- **Torx / Flat-head** – less common but needed for some devices
- **Magnetic tip** – helps prevent dropping/losing screws
- **Extraction tool** – retrieves dropped screws from inside cases

### Network Tools
- **Cable tester** – verifies cable integrity and functionality
- **Crimper** – cuts and terminates network cables

### Organization
- **Parts organizer case** – stores screws, standoffs, adapter cards
- 💡 *"Keeping organized is the top tool of the IT professional."*

## 5. Display Technologies (1.2.12)

### Monitor Technologies

| Technology | Full Name | How It Works | Key Advantage |
|------------|-----------|--------------|---------------|
| **LCD** | Liquid Crystal Display | Two polarizing filters + liquid crystal solution; current aligns crystals to pass/block light | Standard baseline |
| **LED** | Light-Emitting Diode | LCD with LED backlighting | Thinner, lighter, brighter, better contrast, lower power |
| **OLED** | Organic LED | Organic material emits light per pixel individually | Deepest black levels; best contrast |

### Projector Technologies

| Technology | How It Works | Brightness |
|------------|--------------|------------|
| **LCD Projector** | Same LCD tech as monitors | Higher lumens (brighter) |
| **DLP Projector** | Spinning color wheel + array of mirrors (1 mirror = 1 pixel); up to 1024 shades of gray | Lower lumens than LCD |

### Brightness Standards
- **Lumen** – unit of brightness; higher = brighter
- **ANSI Lumens** – standardized brightness measurement (ANSI procedure); allows fair comparison between projectors
- **Color brightness** – measures red, green, and blue output separately

## 6. VR and AR Headsets (1.2.13)

### Virtual Reality (VR)
- Creates a fully **simulated 3D environment**
- Headset **blocks all ambient light** from surroundings
- Tracks user motion and adjusts display in real time
- Hardware options: tethered, standalone, or mobile
- Sensors: motion, external visual positioning, camera, motion tracking, accelerometer, gyroscope, magnetometer

### Augmented Reality (AR)
- **Overlays** digital images/audio onto the **real world** in real time
- Headset allows ambient light (you can still see your surroundings)
- Does **not** always require a headset (e.g., smartphone apps)
- Famous example: **Pokémon GO**
- Other devices: **smart glasses** (lightweight, audience-specific)
- Notable device: **Microsoft HoloLens** (integrated holographic processing unit)

### VR vs AR Comparison

| Feature | VR | AR |
|---------|----|----|
| Ambient light | Blocked | Allowed |
| Environment | Fully virtual | Real world + overlay |
| Hardware | Headset required | Headset, glasses, or phone |
| Key sensors | Motion, gyroscope, accelerometer | Camera, GPS, motion sensors |
| Example | Oculus, PlayStation VR | HoloLens, Pokémon GO |
