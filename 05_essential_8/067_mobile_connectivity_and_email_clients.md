# Mobile Devices & Connectivity — Study Notes

---

## 1. Mobile Device Hardware

- Mobile phones are highly integrated devices — components are packed tightly, making self-repair difficult.
- Most repairs require going through the manufacturer.
- **User-replaceable components:**
  - **SD Card** — expands storage capacity.
  - **SIM Card** — connects the device to a carrier network; stores contacts and messages. Some devices support **Dual SIM** (two numbers on one phone).
  - **Battery** — removable on some models. Check regularly for swelling; avoid exposing the phone to direct sunlight.

## 2. Mobile Connectivity

### Wireless Connection Types

| Type | Notes |
|------|-------|
| **Wi-Fi** | Preferred — usually free, uses less battery than cellular |
| **Cellular** | Uses cell towers and satellites; may incur carrier costs |

### Device Identification Numbers

| Identifier | Description |
|------------|-------------|
| **IMEI** | 15-digit number that uniquely identifies the **device** |
| **IMSI** | Number that identifies the **user/subscriber**, stored on the SIM card |

### Wi-Fi Security Best Practices

- Enable the highest available security protocol (e.g., **WPA2**) on home networks.
- Never enter login credentials over an unencrypted connection.
- Use a **VPN** whenever possible.

### Manually Connecting to Wi-Fi (Android)

1. Settings → Add network
2. Enter the network SSID
3. Select security type
4. Enter password
5. Tap Save

## 3. Cellular Network Generations

| Generation | Era | Speed | Key Features |
|------------|-----|-------|--------------|
| **2G** | 1990s | Up to 1 Mb/s | Introduced SMS (text) and MMS (photo/video) |
| **3G** | Early 2000s | Up to 2 Mb/s | Enabled mobile internet access |
| **4G** | 2010s | 100 Mb/s – 1 Gb/s | Supports gaming; commonly implemented via LTE |
| **5G** | 2019–present | Up to 20 Gb/s | Greater efficiency and speed |

- Mobile phones are **backward compatible** — a 5G phone automatically falls back to 4G when 5G is unavailable.
- Cellular standards are **not uniform worldwide** — some phones only work in specific regions.

---

## 4. Key Wireless Settings

### Airplane Mode
- Disables **all wireless radios**: cellular, Wi-Fi, and Bluetooth simultaneously.
- Wi-Fi or Bluetooth can be re-enabled individually while Airplane Mode is on.

### Cellular Data Toggle
- Enables or disables mobile data independently from Wi-Fi.
- Useful for controlling costs, especially during international travel (roaming).
- **iOS path:** Settings → General → Cellular

### Hotspot
- Allows a cellular device to share its mobile data connection with other devices, acting as a portable Wi-Fi router.

## 5. Email on Mobile

### Components of an Email System

| Component | Role |
|-----------|------|
| **Email Server** | Receives, forwards, and stores emails until retrieved |
| **Email Client** | The app or interface used to compose, read, and manage emails |

### Types of Email Clients

- **Web-based** — accessed through a browser (e.g., Gmail in Chrome); platform-independent.
- **Standalone application** — installed directly on the device (e.g., Outlook, Apple Mail); **platform-dependent**.
